How to Use This Calculator

The fluid type calculator takes in four parameters: minimum and maximum font-size and minimum and maximum viewport width. Values can be entered in either px or rem. From these parameters, a CSS property will be generated that proportionally scales your font size between the specific min and max viewport widths, while never exceeding your min and max font size values.
